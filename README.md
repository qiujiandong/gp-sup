# 毕业设计补充材料

> Graduation Project  Suplimentary Material, GP-SUP

相关材料主要包括四个部分：

- [DSP](DSP)：DSP软件设计
- [Hardware](Hardware)：硬件电路设计原理图
- [Logic](Logic)：FPGA内的逻辑电路设计
- [Linux](Linux)：Linux上位机中的XDMA驱动程序，应用程序与Matlab测试平台
